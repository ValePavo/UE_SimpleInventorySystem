# Simple Inventory System – Unreal Engine 5

This project demonstrates a simple inventory system created entirely with Blueprints in Unreal Engine 5.  
It includes item pickup, dropping items, and basic inventory management mechanics.

## Overview
- Designed as a small, focused example project to showcase gameplay programming skills in UE5.  
- Fully implemented using Blueprints, no C++ required.  
- Clean structure and modular approach for easy extension.

## Features
- Pickup and drop items in the world
- Inventory UI showing collected items
- Modular Blueprint setup for easy adding of new items
- Optimized for simple gameplay testing

## Controls
- **I** – Open inventory  
- **Right-click** – Drop selected item
